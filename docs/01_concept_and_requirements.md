# 📂 01_concept_and_requirements.md (v1.1) - 서비스 기획 및 요구사항

## 1. 서비스 개요 (Service Overview)
* **서비스명**: saju_master
* **한 줄 정의**: 전통 만세력 데이터와 초거대 언어 모델(LLM)을 결합하여, 딱딱한 사주 풀이를 현대적이고 친절한 '나만의 조언'으로 변환해주는 AI 사주 비서 서비스.
* **핵심 가치**: "어려운 한자 나열 대신, 오늘 당신에게 필요한 실질적인 행동 지침을 데이터로 증명합니다."

---

## 2. 서비스 목표 (Service Goals)
1. **데이터 기반 신뢰**: 검증된 만세력 라이브러리를 활용하여 사주 원천 데이터(간지)의 정확성 확보.
2. **개인화된 콘텐츠**: 사용자의 사주와 당일의 기운을 조합하여 매일 다른 맞춤형 운세 생성.
3. **지속적 재방문(Retention)**: 푸시 알림과 데일리 콘텐츠를 통해 사용자가 매일 아침 접속하는 습관 형성.
4. **운영 효율 극대화**: 조립식 문구(Hybrid AI) 구조를 통해 LLM API 호출 비용을 획기적으로 절감.

---

## 3. 타겟 사용자 (Target Audience)
* **2030 MZ 세대**: 어려운 한자 풀이보다 직관적인 점수와 세련된 UI의 운세를 선호하는 층.
* **결정 장애를 겪는 사용자**: 중요한 이직, 투자, 연애 등 선택의 기로에서 심리적 가이드가 필요한 층.
* **지인 중심 사용자**: 나뿐만 아니라 가족, 친구, 연인과의 합이나 운세를 가볍게 관리하고 싶은 층.

---

## 4. 핵심 기능 (Core Features) ### 궁합 추가
1. **AI 사주 정밀 분석**: 생년월일시 기반 만세력 도출 및 성격/특징 AI 해석 리포트.
2. **5대 지표 데일리 운세**: 종합/재물/연애/건강/직장 점수를 시각화(Radar Chart 등)하여 제공.
3. **멀티 프로필 관리**: 최대 5명 이상의 지인 사주 데이터를 등록하고 스위칭하며 확인 가능.
4. **스마트 푸시 알림**: 대표 프로필의 오늘 운세를 요약하여 매일 아침 정해진 시간에 배달.
5. **행운의 아이템 추천**: 당일 사주 구성에 부족한 기운을 채워줄 수 있는 색상이나 소품 추천.
6. **AI 인연 궁합 (New)**: 등록된 프로필 간의 오행 조화를 분석하여 연애/비즈니스/친구 궁합 제공.

---

## 5. 사용자 시나리오 (User Scenario)
1. **최초 접속**: 사용자가 웹/앱 접속 시 비회원 식별용 UUID 발급 및 로컬 저장.
2. **정보 입력**: 본인 이름(닉네임), 생년월일, 태어난 시간을 입력하여 '대표 프로필' 등록.
3. **운세 확인**: AI가 계산된 간지 데이터를 바탕으로 오늘의 5대 지표 점수와 행운의 아이템 제시.
4. **지인 등록**: 친구와의 궁합을 위해 친구의 사주 정보를 추가 등록.
5. **습관 형성**: 다음 날 아침, "오늘 재물운이 100점입니다!"라는 알림을 클릭하여 재접속.

---

## 6. 서비스 차별점 (Differentiation)
* **Hybrid AI 문장 생성**: 100% 실시간 생성 방식이 아닌, 미리 검증된 `fortune_phrases`(인사말, 조언 등)와 AI의 분석 본문을 결합하여 **품질 안정성**과 **비용 절감**을 동시 달성.
* **데이터 일관성 유지**: 동일 날짜 내 재접속 시 DB에 저장된 결과를 노출하여 답변의 번복 방지 및 신뢰도 확보.
* **장벽 없는 UX**: 복잡한 회원가입 없이 즉시 서비스를 이용할 수 있는 비회원 기반의 빠른 접근성.

---

## 7. 수익 모델 (Business Model)
본 프로젝트는 초기 단계에서 LLM 기반 개인화 콘텐츠 생성 시스템의 효율적 설계에 집중하되, 향후 다음과 같은 수익화 전략을 적용한다.

### 1) 프리미엄 심층 리포트 (Micro-payment)
* 기본 제공되는 데일리 운세를 넘어, AI가 사용자의 사주 원국을 정밀 분석한 '월간/연간 종합 리포트' 유료 판매.

### 2) 관계 기반 확장 기능 (Social Expansion)
* 지인 프로필 데이터를 활용한 '1:1 궁합 분석', '오늘의 궁합 랭킹' 등 관계 중심 기능 유료화.

### 3) 하이브리드 광고 모델 (Ad-based Revenue)
* 조립식 문구 구조로 API 토큰 비용을 최소화(Low-cost)하여, 광고 수익만으로도 서비스 운영비(LTV > CAC) 확보 가능.

### 4) 구독형 프리미엄 (Subscription)
* 광고 제거, 무제한 프로필 등록, 구독자 전용 'AI 사주 상담 챗봇' 이용 권한 부여.
